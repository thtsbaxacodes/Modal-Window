# 🪟 Modal Window

A simple modal window project built with **HTML**, **CSS**, and **JavaScript**.

## 🚀 Features

* Open and close the modal
* Close with the overlay
* Close with the Escape key
* Responsive and clean design

## 🛠️ Technologies

* HTML5
* CSS3
* JavaScript (ES6)

## 📖 What I Learned

* DOM manipulation
* Event handling
* Keyboard events
* Class toggling
* JavaScript fundamentals
